## What "eligible for upgrade" means here

None of the 16 upgrade-eligible questions can jump to `course-verified`
(that tier is reserved for confirmation against Vinci's actual manual,
which still doesn't exist for Chapter 3). The realistic upgrade path is
to `source-backed-study` or a documented "class-note-supported,
multiple-source-confirmed" status, with a `sourceReference` citing both
the original quiz batch and this class deck — mirroring exactly how
Chapter 4's quiz-derived questions were upgraded once independently
confirmed by that chapter's manual. This task does not perform that
upgrade (per instruction, no live question was edited); it identifies the
16 candidates precisely so a future promotion task can act on them
directly.
