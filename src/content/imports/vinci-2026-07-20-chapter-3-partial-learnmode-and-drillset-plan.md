## Recommended partial Learn Mode structure

Each topic's content should follow the same shape already used across
this app (`explanation`, `keyFacts`, `confusionTraps`, `examWatchFor`,
`sourceReferences`, `confidence`), with two disciplined rules carried
over from this reconciliation:

1. **Every `keyFacts` entry drawn only from the class deck should be
   tagged at a `needs-review`-equivalent confidence, distinct from
   entries confirmed by CFR/official sources or by a second independent
   source** — mirroring exactly how Chapter 5's guide already
   distinguishes manual-confirmed facts from USCG-source-only facts within
   the same module.
2. **Do not state the corrected painter-length figure and the live
   `q-quiz1a-heaving-line-length`'s heaving-line framing side by side
   without a note** — building Learn Mode content for Automatic Launching
   or Liferaft Stowage should either wait for that question's correction
   or explicitly flag the tension the same way this report does, not
   silently pick one.

## Recommended drill-set groupings (4, once questions are promoted)

Following the established documentation-only convention
(`vinci-chapter-4-drill-sets.md`, `vinci-chapter-5-drill-sets.md`,
`vinci-chapter-6-drill-sets.md`) — not a code change:

1. **Liferaft Equipment & Survival Packs** — diagram identification (all
   3 diagram sources), CFR equipment-list facts, food/water rationing.
2. **Launching & Deployment** — davit/manual/automatic launching,
   inflation mechanism, hydrostatic release, painter/operating cord
   (once the heaving-line question is corrected).
3. **Righting & Boarding** — capsized-raft righting, boarding from vessel
   and from water, assisting an injured survivor.
4. **Design, Stowage & Markings** — construction/major parts, stowage
   requirements, markings, and maintenance/inspection.

This is a natural fit for Chapter 3's current shape: a 4-set structure
matches the chapter's overall size (36 live questions today, likely
50+ after this cycle's candidates promote) better than Chapter 4's
5-set structure, which was sized for a much larger question pool.

## Recommended future Study Guide structure (not built in this task)

When a Chapter 3 Study Guide is eventually built, it should mirror the
drill-set clusters above plus a dedicated comparison-table section for
the confusion risks this cycle surfaced:

  here (that's Chapter 4's territory) — instead, Chapter 3's natural
  comparison table is **lifeboat sea painter vs. liferaft painter/
  operating cord** (two different formulas for two different craft,

  drift-slowing overlap flagged in the revised question audit.
